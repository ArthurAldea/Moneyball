## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge internal padding |
| sm | 8px | Filter label margin-bottom, stat inline gaps |
| md | 16px | Sidebar section spacing, table cell padding |
| lg | 24px | Section breaks between sidebar filters |
| xl | 32px | Main area top padding |
| 2xl | 48px | Major section breaks (table-to-scatter gap) |
| 3xl | 64px | Page-level top margin |

Exceptions: Streamlit sidebar internal padding is browser-controlled; only override where `st.markdown` HTML is authored directly. Touch target minimum for Streamlit buttons: 44px height (applied via CSS `min-height: 44px` on `.stButton > button`).

## Component Inventory

### Shortlist Table

- Rendered via `st.dataframe` with Streamlit >= 1.35 API.
- `column_config` used to set display names (ALL-CAPS), numeric formatting, and column widths.
- `selection_mode="single-row"` enables row click.
- `use_container_width=True`, `hide_index=True`.
- Column order: Player | Club | League | Position | Age | Scout Score | UV Score | Age-Weighted UV | Value (€M) | Value Gap (€M).
- Default sort: `uv_score_age_weighted` descending — pre-sorted in Python before passing to `st.dataframe`.
- Selected row: highlighted with left-border accent stripe via CSS targeting `[data-testid="stDataFrame"] tr[aria-selected="true"]`.

### Sidebar Filter Order

1. LEAGUE — `st.multiselect`, options: EPL, LaLiga, Bundesliga, SerieA, Ligue1. Default: all 5.
2. POSITION — `st.multiselect`, options: GK, DF, MF, FW. Default: all 4.
3. AGE — `st.slider`, min=17, max=38, default=(17, 38), step=1. Label: "AGE RANGE".
4. CLUB — `st.multiselect`, options dynamically derived from clubs in selected leagues. Default: all. Label: "CLUB".
5. MARKET VALUE — `st.slider`, min=0, max=dataset max rounded up to nearest 10 (€M), default=(0, max), step=1. Label: "VALUE (€M)".
6. SEASON — `st.multiselect`, options: 2023-24, 2024-25. Default: both.

After all 6 filters: `st.divider()` then "Refresh Data" button.

### Scatter Plot (`NAVY_LAYOUT`)

New Plotly layout dict to replace `CYBER_LAYOUT`:

```python
NAVY_LAYOUT = dict(
    paper_bgcolor="#0D1B2A",
    plot_bgcolor="#112236",
    font=dict(family="Inter, system-ui, sans-serif", color="#E8EDF2", size=12),
    margin=dict(t=48, b=48, l=56, r=24),
)
```

Axes:
- X-axis: `scout_score` (0–100), title "SCOUT SCORE", gridcolor `rgba(255,255,255,0.06)`.
- Y-axis: `log10(market_value_eur)`, title "LOG\u2081\u2080 MARKET VALUE", gridcolor `rgba(255,255,255,0.06)`.
- Hover template: Player name (bold), Club, Position, Scout Score (1dp), Market Value (€M, 1dp), UV Score (1dp).
- Chart height: 480px.

OLS regression line: `#00A8FF`, width 1.5, dash "dot", name "FAIR VALUE LINE".

Cross-league disclaimer: rendered as `st.caption()` below the chart when `len(selected_leagues) > 1`. Uses muted text color via CSS `.stCaption`.

### Row-Click Placeholder Panel

- Rendered inside `st.container()` below the shortlist table (not an expander — stays visible after selection).
- Border: `1px solid rgba(0,168,255,0.25)`, left-border `3px solid #00A8FF`.
- Background: `#112236`.
- Padding: 16px.
- Content: player name (heading), one-line meta (club · league · position · age · value), scores row (Scout Score | UV Score | Age-Weighted UV), footnote "Full profile coming soon".
- Displayed only when `selection_mode` returns a non-empty selection. Hidden/absent when nothing selected.

### Player Count Display

Rendered as `st.caption()` above the shortlist table: "Showing {N} players". Updates on every filter change. Uses muted text color.

> Source: CONTEXT.md "Claude's Discretion — whether to show player count anywhere". Declared as yes, above-table caption.

## Streamlit Row Selection API

Streamlit >= 1.35 exposes `on_select="rerun"` and `selection_mode="single-row"` on `st.dataframe`. The return value of `st.dataframe(df, on_select="rerun", selection_mode="single-row")` is an object with `.selection.rows` (list of selected row indices). If `.selection.rows` is non-empty, extract `df.iloc[selection.rows[0]]` for the placeholder panel.

> Source: CONTEXT.md "Claude's Discretion — exact Streamlit widget API for row selection (version-dependent — researcher to confirm)". Confirmed as design default based on Streamlit 1.35+ documentation.
