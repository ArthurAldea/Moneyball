## What Was Built

- `_do_playwright_get(url: str) -> str`: thin wrapper that launches headless Chromium via `sync_playwright`, navigates to URL, waits for a `<table>` element, and returns full page HTML string
- `_playwright_fetch(url: str) -> str`: retry loop around `_do_playwright_get` using `FBREF_BACKOFF_SEQUENCE`; detects Cloudflare challenge via `"Just a moment"` / `"cf-browser-verification"` in returned HTML; raises `RuntimeError` after 3 retries
- Removed `_fetch_with_backoff` entirely
- Updated both call sites: `scrape_fbref_stat` and `scrape_fbref_standings`
- Added `playwright` to `requirements.txt`
